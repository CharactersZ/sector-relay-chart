# 快速开始指南

## 第一步：安装依赖

```bash
pip install -r requirements.txt
```

## 第二步：运行脚本

最简单的使用方式，直接运行：

```bash
python sector_relay_chart.py
```

这会自动：
- 获取今天涨幅前30的行业板块
- 按涨幅从高到低排序
- 更新到Excel文件 `赛道接力图.xlsx`
- 为每个板块分配固定颜色

## 第三步：查看结果

打开生成的 `赛道接力图.xlsx` 文件，你会看到：
- 第一行：日期（从左到右按时间排序）
- 第一列：排名（1, 2, 3...）
- 每个单元格：板块名称，带有固定颜色

## 每天更新

每天收盘后（15:00之后）运行一次：

```bash
python sector_relay_chart.py
```

或者设置定时任务自动运行（见README.md）。

## 常用命令

```bash
# 查看已有数据
python sector_relay_chart.py --view

# 更新指定日期
python sector_relay_chart.py -d 2026-01-29

# 自定义取前20个板块
python sector_relay_chart.py -n 20

# 自动覆盖模式（适合定时任务）
python sector_relay_chart.py --auto-overwrite
```

## 文件说明

- `赛道接力图.xlsx` - 主Excel文件，包含所有历史数据
- `sector_colors.json` - 板块颜色映射，确保同一板块颜色一致（不要手动修改）

## 注意事项

1. **运行时间**：建议在交易日收盘后运行，确保数据完整
2. **网络要求**：需要网络连接获取实时数据
3. **数据来源**：使用akshare库，数据来自东方财富等财经网站
