# 打包快速参考

## ❓ 在Mac上能打包Windows exe吗？

**简短回答**：不能直接打包，但有几种解决方案。

## ✅ 推荐方案

### 方案1：在Windows电脑上打包（最简单）

1. 将项目文件复制到Windows电脑
2. 运行 `build_windows.bat`
3. 完成！

### 方案2：使用GitHub Actions（自动化）

1. 创建GitHub仓库
2. 推送代码
3. 在GitHub Actions页面手动触发构建
4. 下载打包好的exe

**优点**：在Mac上操作，自动在Windows环境打包

## 📁 已创建的文件

- `build_windows.bat` - Windows打包脚本（在Windows上运行）
- `build.spec` - PyInstaller配置文件
- `build_mac.sh` - Mac打包脚本（只能打包Mac版本）
- `.github/workflows/build.yml` - GitHub Actions自动打包配置
- `打包说明.md` - 详细打包说明

## 🚀 快速开始

### 如果你想在Windows上打包：

```cmd
# 1. 复制文件到Windows
# 2. 安装Python和依赖
pip install -r requirements.txt
pip install pyinstaller

# 3. 运行打包脚本
build_windows.bat
```

### 如果你想用GitHub Actions：

1. 创建GitHub仓库
2. 推送代码
3. 在GitHub网页：Actions → Build Windows EXE → Run workflow
4. 完成后下载exe文件

## 📝 注意事项

- Mac上**不能直接**打包Windows exe
- PyInstaller不支持跨平台打包
- 需要在目标平台（Windows）上打包，或使用CI/CD服务
